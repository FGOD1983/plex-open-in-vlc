Open files in VLC from the Plex web interface

This bookmarklet copies the download URL to clipboard - for easy usage with a download manager.

Based on and forked from: https://piplong.run/plxdwnld/.

How to install

  1. Install vlc:// protocol handler: https://github.com/stefansundin/vlc-protocol
  2. Drag this button to your bookmarks bar: Open In VLC

How to use

  1. Open the Plex web interface and browse to the details page of a media item. The bookmarklet currently works for Movies and TV Shows.
  2. Click the bookmark button to open the file in VLC.
